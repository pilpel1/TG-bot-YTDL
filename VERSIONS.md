  - הודעת הסבר אחידה לכל סוגי ההודעות שאינן מכילות קישור תקין

### גרסה 0.4.3

### גרסה 0.4.4

### גרסה 0.5.0 🚀
**תמיכה בקבצים גדולים עד 2GB!**
- **Local Bot API Server integration**: אפשרות לשליחת קבצים עד 2GB במקום מגבלת 50MB
- **שני מצבי הפעלה**:
  - מצב פשוט: 50MB עם Telegram Bot API רגיל
  - מצב מתקדם: 2GB עם Local Bot API Server
- **תמיכה טכנית מלאה**:
  - WSL2 integration for Windows
  - Docker containerization for local server
  - Environment variables management
  - Automated setup scripts (.bat files)
- **הוראות התקנה מעודכנות**:
  - הפרדה ברורה בין מצב פשוט למתקדם
  - הוראות מפורטות לכל מערכת הפעלה
  - הדגשת חשיבות FFmpeg
- **שיפורי תיעוד**: README.md מעודכן עם הוראות שלב אחר שלב

### גרסה 0.5.1 🔧

  - טיפול משופר בשגיאות רשת

### גרסה 0.5.2 🎯
**שיפורי ממשק וארגון פרויקט**
- **שינוי שמות סקריפטים**:
  - `run_bot.bat` → `run_bot_simple_50MB.bat`
  - `run_bot_with_local_server.bat` → `run_bot_advanced_2GB.bat`
  - שמות מקבילים בלינוקס (.sh)
  - שמות אינטואיטיביים המשקפים את תפקוד הסקריפט
- **הוספת פקודה חדשה `/mode`**:
  - הצגת מצב הבוט הנוכחי (50MB או 2GB)
  - הסבר על המצבים השונים
  - הדרכה על החלפת מצבים
- **שיפור התנהגות חלונות CLI**:
  - חלונות השרת והמנהל מתמזערים אוטומטית לאחר הפעלה מוצלחת
  - רק חלון הבוט נשאר פתוח עבור הלוגים
  - שיפור חוויית המשתמש
- **עדכוני תיעוד**:
  - תיאורים מדויקים יותר של מצבי ההפעלה
  - הדגשת ה-fallback mechanism במצב המתקדם
  - עדכון README.md ו-scripts/README.md עם השמות החדשים

### גרסה 0.5.3 🎵
**תיקון באג הורדת אודיו ושיפורים טכניים**
- **תיקון באג קריטי בהורדת אודיו**:
  - הבוט הוריד קבצי MP4 מלאים (וידאו) במקום אודיו בלבד
  - קבצים נשלחו כ"אודיו" בטלגרם אך הכילו וידאו מלא
  - כעת: חילוץ אודיו אמיתי באמצעות FFmpeg
- **אינטגרציה עם FFmpeg**:
  - זיהוי אוטומטי של FFmpeg בהפעלת הבוט
  - חילוץ אודיו מסרטונים (M4A/AAC/MP3)
  - הפחתת גודל קבצים פי 5-6 בהורדות אודיו
  - עבודה חלקה גם ללא FFmpeg (עם אזהרות)
- **שיפורי בחירת פורמט YouTube**:
  - תיקון דריסת פורמט אודיו על ידי הגדרות YouTube
  - fallback חכם לפורמטי אודיו בלבד כשמבקשים אודיו
  - טיפול משופר בבעיות nsig/SABR של YouTube
- **שיפורי תיעוד**:
  - הוראות התקנה מפורטות של FFmpeg (Windows/Linux/Mac)
  - הסבר על חשיבות FFmpeg והשפעתו על הורדות
  - הסבר טכני על nsig ו-SABR של YouTube
  - סעיף חדש "איך הורדת אודיו עובדת?"
- **שיפורי חוויית משתמש**:
  - תיקון תצוגת אימוג'ים בסקריפטי Windows (החלפה ב-ASCII)
  - הודעת FFmpeg מופיעה רק בהפעלה ולא בכל הורדה
  - לוג מפורט יותר על חילוץ אודיו

## גרסה נוכחית

### גרסה 0.7.0 (נוכחי) 🚀
**שדרוג משמעותי לזרימת הורדות YouTube**
- **תיקון בחירת איכות ב-YouTube**:
  - האיכות שהמשתמש בוחר משפיעה בפועל על הפורמט שיורד
  - הוסרה דריסת הפורמט שגרמה להורדות באותה איכות למרות בחירה שונה
  - שופר fallback לפורמטים זמינים
- **אפשרויות הורדה דינמיות לסרטון בודד**:
  - הצגת איכויות זמינות בפועל לפי הסרטון
  - הצגת גודל משוער ליד כל אפשרות
  - סימון האיכות הטובה ביותר שמתאימה למגבלת הגודל
  - סימון איכויות חסומות עקב משקל גבוה מדי
  - הוספת "אודיו בלבד" לאותה רשימת אפשרויות
- **שיפורי חוויית משתמש**:
  - popup ברור כשלוחצים על איכות שחסומה עקב גודל
  - הוספת כפתור ביטול במסכי הבחירה
  - אפשרות לחזור מיד למצב נקי ולשלוח קישור חדש
- **שיפורים בפלייליסטים של YouTube**:
  - זרימת בחירה נפרדת לפלייליסטים
  - אפשרויות ברורות: איכות גבוהה, רגילה, נמוכה או אודיו בלבד
  - הסבר שההגדרה שתיבחר תחול אוטומטית על כל הסרטונים בפלייליסט
- **שיפורים טכניים להורדות גדולות**:
  - בדיקת גודל משוער מראש וחסימה מוקדמת של קבצים גדולים מדי כשאפשר לדעת את הגודל
  - העלאת timeouts לשליחת קבצים גדולים
  - שיפור הספירה והסיכום בהורדות פלייליסטים
